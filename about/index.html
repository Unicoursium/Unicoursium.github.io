关于EternityX

世间本无永恒，与你在一起的回忆却永远不会忘记

EternityX的前身是Unico Server, 意思是腐竹英文名Unico 的服务器 直到UnicoServer 7的时候 我的好朋友们提议了永恒服务器 这成为了EternityX的来源

从Unico Server 1 一直到Unico Server 9，服务器诞生了很多的玩法 曾经有过起床战争服务器 也拥有了一些玩家们
